# Kubernetes 安全工程师：从基础到专家 完整教材

## 前言

### 本书定位

本教材面向希望系统学习 **Kubernetes 安全** 的技术人员，覆盖从 Linux 基础到云原生安全架构设计的完整知识链路。无论你是传统运维、开发人员，还是网络安全从业者，都可以通过本教材建立扎实的 K8s 安全能力体系。

### 学习路径总览

```
┌─────────────────────────────────────────────────────────────┐
│  第一阶段：基础打底（1-2个月）                                │
│  Linux系统 → Docker容器 → Kubernetes基础概念                │
├─────────────────────────────────────────────────────────────┤
│  第二阶段：进阶管理（2-3个月）                                │
│  K8s集群部署 → 工作负载管理 → 存储网络 → 故障排查           │
│  目标：通过 CKA 认证                                         │
├─────────────────────────────────────────────────────────────┤
│  第三阶段：安全专项（2-3个月）                                │
│  平台安全 → 应用安全 → 运行时安全 → 供应链安全              │
│  目标：通过 CKS 认证                                         │
├─────────────────────────────────────────────────────────────┤
│  第四阶段：专家进阶（持续）                                   │
│  DevSecOps → 云原生攻防 → eBPF → 安全架构设计               │
│  目标：云原生安全架构师 / 安全研究员                         │
└─────────────────────────────────────────────────────────────┘
```

### 各阶段能力要求

| 阶段 | 知识深度 | 实操能力 | 认证目标 |
|------|---------|---------|---------|
| 基础 | 理解原理 | 能搭建实验环境 | - |
| 进阶 | 掌握配置 | 能管理生产集群 | CKA |
| 安全 | 精通机制 | 能加固集群防御 | CKS |
| 专家 | 架构设计 | 能开发安全工具 | - |

### 实验环境说明

**最低配置要求：**
- CPU：4核+
- 内存：8GB+
- 磁盘：50GB+
- 系统：Ubuntu 22.04 LTS / macOS / WSL2

**推荐工具：**
- Docker Desktop（带K8s）或 minikube
- kubectl
- Helm
- VS Code + YAML/Kubernetes 插件
