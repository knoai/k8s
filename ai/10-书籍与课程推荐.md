### 5.3 本地 K8s 搭建

```bash
# kind（推荐，最像真实集群）
kind create cluster --name k8s-ai --config kind-config.yaml

# kind 配置（多节点 + GPU）
# kind-config.yaml
cat <<EOF > kind-config.yaml
kind: Cluster
apiVersion: kind.x-k8s.io/v1alpha4
nodes:
  - role: control-plane
    extraMounts:
      - hostPath: /dev/null
        containerPath: /var/run/nvidia-container-devices/all
  - role: worker
    extraMounts:
      - hostPath: /usr/local/cuda
        containerPath: /usr/local/cuda
EOF

# minikube（GPU 支持较好）
minikube start --driver=docker --gpus all --nodes 3

# k3s（轻量，适合边缘）
curl -sfL https://get.k3s.io | INSTALL_K3S_EXEC="server --docker" sh -

# k3d（k3s in Docker）
k3d cluster create k8s-ai --agents 2 --gpus 1
```

├── 实践: Killer.sh 模拟题

└── 进阶: Programming Kubernetes

├── GPU: NVIDIA 官方文档
├── Kubeflow: 官方文档 + 实验
├── KServe: 官方文档
└── MLflow: 官方 Quickstart

第 6-8 月（MLOps）
├── MLOps: Designing ML Systems

├── 分布式: Ray 官方教程
└── 推理: vLLM 文档

第 9-12 月（专家）
