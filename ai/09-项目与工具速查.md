## 十一、选型速查表

### 推理引擎选型

| 条件 | 推荐 |
|------|------|
| NVIDIA GPU + 追求极致性能 | TensorRT-LLM |
| 通用生产环境（多 GPU） | vLLM |
| 需要 Structured Generation | SGLang |
| HuggingFace 生态深度用户 | TGI |
| 国产 GPU / 特定优化 | LMDeploy |
| CPU / 边缘设备 | llama.cpp |
| 快速本地测试 | Ollama |

### 调度器选型

| 条件 | 推荐 |
|------|------|
| 需要 Gang Scheduling | Volcano |
| K8s 原生队列管理 | Kueue |
| GPU 共享/虚拟化 | HAMi + 默认调度器 |
| 节点弹性扩缩 | Karpenter |
| 大数据 + AI 统一 | Yunikorn |

### MLOps 平台选型

| 条件 | 推荐 |
|------|------|
| 全开源、全功能 | Kubeflow |
| 轻量快速开始 | MLflow + 自建 Pipeline |
| 企业级支持 | OpenShift AI / SageMaker |
| 强工作流需求 | Flyte / Airflow |
| LLM 专项 | AIBrix + vLLM |
