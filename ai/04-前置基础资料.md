## 二、容器技术（1-2 周）

### 2.1 Docker 核心概念

```
Image（镜像）→ 只读模板，分层存储
   │
   ▼ docker run
Container（容器）→ 镜像的运行实例，可写层
   │
   ▼ docker commit
New Image（新镜像）→ 保存容器状态
```

### 2.2 Dockerfile 最佳实践

```dockerfile
# 多阶段构建示例：Python ML 应用
FROM python:3.11-slim as builder

WORKDIR /app
COPY requirements.txt .
RUN pip install --user --no-cache-dir -r requirements.txt

# 生产镜像
FROM python:3.11-slim

# 安全：非 root 运行
RUN groupadd -r appuser && useradd -r -g appuser appuser

WORKDIR /app
COPY --from=builder /root/.local /home/appuser/.local
COPY . .

ENV PATH=/home/appuser/.local/bin:$PATH
USER appuser

EXPOSE 8000
HEALTHCHECK --interval=30s --timeout=3s \
    CMD curl -f http://localhost:8000/health || exit 1

CMD ["python", "app.py"]
```

**最佳实践清单**：
- [ ] 使用多阶段构建减小镜像体积
- [ ] 使用具体版本标签（不用 `latest`）
- [ ] 非 root 运行容器
- [ ] `.dockerignore` 排除不必要文件
- [ ] 合理利用缓存层（不频繁变动的放前面）
- [ ] 只安装生产依赖（`--no-dev`）

### 2.5 容器安全

```bash

  --read-only \                    # 只读根文件系统
  --user 1000:1000 \               # 非 root
```

### 3.3 异步编程（高并发必备）

```python
import asyncio
import aiohttp

async def fetch_url(session: aiohttp.ClientSession, url: str) -> str:
    async with session.get(url) as response:
        return await response.text()

async def main():
    urls = ["http://api1", "http://api2", "http://api3"]
    async with aiohttp.ClientSession() as session:
        tasks = [fetch_url(session, url) for url in urls]
        results = await asyncio.gather(*tasks)
        return results

# 运行
asyncio.run(main())
```

## 四、Go 语言（2-3 周，强烈推荐）

### 4.1 为什么必须学 Go？

K8s 全部用 Go 编写，Kubeflow、Volcano、KServe、etcd、Prometheus 等核心组件也是 Go。要读懂源码、参与开源、二次开发，Go 是必经之路。

### 4.2 Go 核心特性

```go
package main

import (
    "context"
    "fmt"
    "time"
)

// 接口：Go 的核心抽象
type Runner interface {
    Run(ctx context.Context) error
}

// 结构体
type Trainer struct {
    Name   string
    GPUCount int
    config Config
}

// 方法
func (t *Trainer) Run(ctx context.Context) error {
    // 使用 context 处理超时和取消
    select {
    case <-ctx.Done():
        return ctx.Err()
    case <-time.After(10 * time.Second):
        fmt.Printf("%s training complete\n", t.Name)
        return nil
    }
}

// 并发：goroutine + channel
func main() {
    jobs := make(chan int, 10)
    results := make(chan string, 10)
    
    // 启动 3 个 worker
    for w := 1; w <= 3; w++ {
        go worker(w, jobs, results)
    }
    
    // 发送 5 个任务
    for j := 1; j <= 5; j++ {
        jobs <- j
    }
    close(jobs)
    
    // 收集结果
    for a := 1; a <= 5; a++ {
        fmt.Println(<-results)
    }
}

func worker(id int, jobs <-chan int, results chan<- string) {
    for j := range jobs {
        time.Sleep(time.Second)
        results <- fmt.Sprintf("worker %d processed job %d", id, j)
    }
}
```

### 4.4 Go 测试

```go
// table-driven tests（表驱动测试）
func TestCalculate(t *testing.T) {
    tests := []struct {
        name     string
        input    int
        expected int
        wantErr  bool
    }{
        {"positive", 5, 25, false},
        {"zero", 0, 0, false},
        {"negative", -1, 0, true},
    }
    
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            got, err := Calculate(tt.input)
            if (err != nil) != tt.wantErr {
                t.Errorf("Calculate() error = %v, wantErr %v", err, tt.wantErr)
                return
            }
            if got != tt.expected {
                t.Errorf("Calculate() = %v, want %v", got, tt.expected)
            }
        })
    }
}

// benchmark
func BenchmarkCalculate(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Calculate(100)
    }
}
```
